# Data-Professional-Survey-Analysis
This Power BI dashboard dives into insights from a global survey of 630 data professionals, exploring salaries, job roles, programming language preferences, and career challenges. It helps paint a clear picture of the data industry and what professionals experience in their careers.

## Analysis Questions
How many professionals participated in the survey?

What is the average age and salary by job role?

Which programming language is most popular?

Which countries have the most survey takers?

How satisfied are professionals with work-life balance and salary?

How hard is it to break into data roles?

## Process
Collected survey responses from 630 participants worldwide.

Cleaned and standardized data for job titles, salaries, and languages.

Built a structured data model with relationships between roles, countries, and demographics.

Created DAX measures for key metrics like average salary, age, and satisfaction scores.

Designed interactive visuals for salary trends, language popularity, and career difficulty.

## Insights

*Python dominates as the favorite programming language among data professionals.

*Data Scientists and Architects earn the highest salaries; students and entry-level roles earn the least.

*United States and India have the largest share of respondents.

*Professionals are moderately happy with work-life balance but less satisfied with salary.

*Breaking into data roles is considered challenging by most respondents.

## Final Conclusion
The survey shows that data careers are rewarding but competitive. Python remains the most in-demand skill, and salaries vary widely by role. While work-life balance is acceptable, salary satisfaction and entry barriers need attention. For aspiring professionals, focusing on Python and core data skills is key to success.
